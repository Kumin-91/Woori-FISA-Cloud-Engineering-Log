# Install MySQL on Linux

운영체제 패키지 관리자를 통해 MySQL 8.0 서버를 설치하고 실행 상태를 확인합니다.

## 1. MySQL 설치 과정

### 1.1 패키지 업데이트

```Bash
sudo apt update
```

### 1.2 MySQL 설치

```Bash
sudo apt install mysql-server
```

### 1.3 MySQL 설치 확인

```Bash
mysql --version
```

```Plain text
mysql  Ver 8.0.44-0ubuntu0.24.04.2 for Linux on aarch64 ((Ubuntu))
```

### 1.4 MySQL 실행 상태 확인

```Bash
sudo systemctl status mysql
```

```Plain text
● mysql.service - MySQL Community Server
     Loaded: loaded (/usr/lib/systemd/system/mysql.service; enabled; preset: enabled)
     Active: active (running) since Sun 2026-01-04 20:35:15 KST; 18min ago
    Process: 677 ExecStartPre=/usr/share/mysql/mysql-systemd-start pre (code=exited, status=0/SUCCESS)
   Main PID: 849 (mysqld)
     Status: "Server is operational"
      Tasks: 38 (limit: 4546)
     Memory: 426.5M (peak: 439.4M)
        CPU: 15.582s
     CGroup: /system.slice/mysql.service
             └─849 /usr/sbin/mysqld
```

* `Active: active (running)` 상태가 아니라면 아래 명령어로 서비스를 실행합니다.

```Bash
#서비스 즉시 실행
sudo systemctl start mysql
#시스템 재부팅 시 서비스 자동 실행 활성화
sudo systemctl enable mysql
```

## 2. MySQL 접속 테스트

### 2.1 root 계정으로 접속

```Bash
sudo mysql -u root -p
```

* 비밀번호를 묻는 프롬프트에서 엔터 (빈칸 입력)

### 2.2 databse 확인

```SQL
show databases;
```

```Plain text

+--------------------+
| information_schema |
| mysql              |
| performance_schema |
| sys                |
```

```SQL
select host, user from mysql.user;
```
