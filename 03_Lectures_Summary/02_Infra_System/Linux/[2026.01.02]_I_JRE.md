# JAR Packaging and Execution

## 1. JAR Packaging & Transfer

로컬 개발 환경(macOS)에서 결과물을 패키징하고 서버로 전달하는 과정입니다.

### 1.1 JAR 파일 생성 (Visual Studio Code)

1. `cmd + shift + p`

2. `Java: Export jar`

3. Main Class 선택

4. `step06_jarTest.jar` (`프로젝트명.jar`) 생성 확인

### 1.2 로컬 환경 JAR 테스트

```Bash
java -jar step06_jarTest.jar
```

```Plain text
1
```

* `no main manifest attribute` 에러가 발생한다면, JAR 패키징 시 Main Class가 정확히 선택되었는지 다시 확인

### 1.3 Linux 서버로 전송

* Windows: MobaXterm 드래그 앤 드롭 전송

* macOS: scp 명령어 혹은 Termius, Cyberduck 등 활용

```Bash
#파일이 있는 디렉토리에서 실행
scp -P 2201 step06_jarTest.jar ubuntu@127.0.0.1:/home/ubuntu
```

## 2. Execution JAR on Linux

운영 환경인 리눅스에서 프로그램을 실행하는 과정입니다.

### 2.1 JRE 설치

```Bash
sudo apt install openjdk-17-jre-headless
```

* 설치 확인

```Bash
java -version
```

```Plain text
openjdk version "17.0.17" 2025-10-21
OpenJDK Runtime Environment (build 17.0.17+10-Ubuntu-124.04)
OpenJDK 64-Bit Server VM (build 17.0.17+10-Ubuntu-124.04, mixed mode, sharing)
```

### 2.2 JAR 파일 실행

```Bash
java -jar step06_jarTest.jar
```

```Plain text
1
```
