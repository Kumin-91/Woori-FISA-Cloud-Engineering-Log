## 2. 타겟 테이블(Target Table) 설계

* Oracle

    ```SQL
    CREATE TABLE merchandise_sales (
    purchase_date DATE,             -- 구매 날짜
    product_name  VARCHAR2(500),    -- 상품명
    quantity      NUMBER(10),       -- 수량
    price         NUMBER(10)        -- 가격
    );
    ```

* MySQL

    ```SQL
    CREATE TABLE merchandise_sales (
    purchase_date DATE,             -- 구매 날짜
    product_name  VARCHAR(500),     -- 상품명
    quantity      INT,              -- 수량
    price         INT               -- 가격
    );
    ```
