    ```SQL
    SELECT * FROM tab;
    ```

* MySQL

    ```SQL
    -- 실습용 데이터베이스 선택
    USE fisa;
    ```

## 2. 특정 table에서 모든 내용 검색

* Oracle & MySQL

    ```SQL
    SELECT * FROM emp;
    ```

## 3. 특정 table의 구조 검색 (Metadata)

* Oracle & MySQL

    ```SQL
    -- Describe의 약어
    DESC emp;
    ```

    * DBeaver 등 일부 GUI 툴의 Oracle 환경에서는 DESC 명령어가 지원되지 않음

## 4. 특정 컬럼 (Column) 검색

* Oracle & MySQL

    ```SQL
    -- 1개 컬럼 검색
    SELECT empno FROM emp;
    -- 2개 이상 컬럼 검색
    SELECT empno, ename FROM emp;
    -- AS (Alias)로 컬럼 별칭 지정
    -- 공백이나 특수문자 포함 시 큰따옴표 사용
    SELECT empno AS "사번", ename AS "이름" FROM emp;
    ```

## 5. 중복된 항목을 제외하고 컬럼 검색

* Oracle & MySQL

    ```SQL
    SELECT DISTINCT deptno FROM emp;
    ```

    * 정렬 `ORDER BY` 결과의 정렬 순서 (오름차순/내림차순)는 보장되지 않음

## 6. 오름차순 / 내림차순 정렬

* 실행 순서: `FROM` -> `SELECT` -> `ORDER BY`

* `SELECT`에서 선택하지 않은 컬럼을 기준으로도 정렬이 가능

* Oracle & MySQL

    ```SQL
    -- 오름차순 (기본값)
    SELECT DISTINCT ename, deptno FROM emp
    ORDER BY empno ASC;
    -- 내림차순
    SELECT DISTINCT ename, deptno FROM emp
    ORDER BY empno DESC;
    ```

## 7. `Date` 타입의 정렬

* Oracle & MySQL

    ```SQL
    -- 오름차순: 오래된 날짜부터 정렬
    SELECT hiredate FROM emp
    ORDER BY hiredate ASC;
    ```

## 8. SELECT 절 내 산술 연산

* Oracle & MySQL

    ```SQL
    SELECT ename, (sal * 12) AS "연봉" FROM emp;
    ```

## 9. 결측치 고려 연산 (NULL Handling)

* 개념 정리

    * 결측치 (Missing Value): 데이터가 없는 상태 (NULL)

    * 이상치 (Outlier): 일반적인 데이터 범위에서 크게 벗어난 값

* comm이 NULL이면 0으로 치환하여 연봉 계산    

    * Oracle: `NVL(컬럼, 치환값)`

        ```SQL
        SELECT ename, sal, (sal * 12) + NVL(comm, 0) AS "연봉" FROM emp;
        ```

    * MySQL: `IFNULL(컬럼, 치환값)`

        ```SQL
        SELECT ename, sal, (sal * 12) + IFNULL(comm, 0) AS "연봉" FROM emp;
        ```
